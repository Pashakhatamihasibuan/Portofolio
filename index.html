Tentang Saya

Saya adalah seorang lulusan Sistem informasi dari Universitas Malikussaleh pada tahun 2023. Saya memiliki pengalaman bekerja sebagai administrasi sekolah, di mana saya terbiasa dengan manajemen data, perhatian terhadap detail, dan problem-solving dalam pekerjaan sehari-hari. Namun, saya selalu tertarik dengan teknologi, khususnya dalam pengembangan web. Karena itu, saya memutuskan untuk beralih ke FrontEnd Development. Saya sudah mempelajari Flutter, JavaScript, dan Tailwind CSS, serta memiliki sertifikasi Flutter Developer. Saya percaya bahwa pengalaman administrasi saya akan mendukung saya dalam berkarir sebagai FrontEnd Developer, terutama dalam hal ketelitian dan efisiensi kerja.
